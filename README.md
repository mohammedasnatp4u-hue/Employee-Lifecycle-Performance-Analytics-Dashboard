IBM Employee Dataset – HR Analytics Dashboard


This project is based on the IBM Employee Dataset and focuses on analyzing workforce data using interactive visualizations. The dashboard provides valuable insights into employee demographics, job satisfaction, education background, and overall workforce trends.

Objectives:
 To analyze employee distribution across different education fields and job roles
 To evaluate job satisfaction based on marital status and other factors
 To understand key HR metrics such as employee count, monthly income, and job levels
 To identify patterns influencing employee performance and retention

Key Features:
 Visual representation of employee data using charts and graphs
 Analysis of education fields, marital status, and demographics (age & gender)
 Insights into job level, hourly rate, and environmental satisfaction
 Interactive filters for better data exploration

Tools Used:
 Power BI / Excel (for data visualization)
 IBM HR Analytics Dataset

Conclusion:
This dashboard helps transform raw employee data into meaningful insights, enabling HR professionals to make data-driven decisions for improving employee satisfaction, retention, and organizational performance.
